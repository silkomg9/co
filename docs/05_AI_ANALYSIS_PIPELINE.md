# AI Analysis Pipeline

## Flow

1. Extract document text
2. Analyze contest purpose
3. Extract evaluation criteria
4. Infer important directions
5. Diagnose user idea
6. Generate questions
7. Create plan

## AI Rules

DO:
- Explain reasoning
- Show evidence
- Ask missing information

DO NOT:
- Promise selection
- Invent achievements
- Create fake numbers

## Output

Structured JSON response recommended.
