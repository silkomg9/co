# Gongmo Coach AI - Project Context

## Overview
공고문을 분석하고 사용자의 아이디어를 공모 취지에 맞는 사업기획으로 발전시키는 AI 코칭 웹앱.

## Goal
- 공고문 이해
- 평가 기준 분석
- 사업 방향 도출
- 아이디어 진단
- 사업계획서 초안 생성

## MVP Flow
공고문 업로드 → AI 분석 → 아이디어 입력 → 코칭 질문 → 사업계획서 생성

## Principles
- AI는 선정 가능성을 보장하지 않는다.
- 근거 기반 분석을 제공한다.
- 사용자가 입력하지 않은 정보는 생성하지 않는다.
