# Product Requirements

## Core Features

### Authentication
- Firebase Authentication
- Email login
- Google login

### Project Management
사용자는 공모사업 프로젝트를 생성하고 관리한다.

### Notice Analysis
입력:
- PDF
- DOCX
- TXT
- URL

출력:
- 사업 목적
- 지원 대상
- 평가 기준
- 예산 조건
- 주요 키워드

### Idea Coaching
AI가 부족한 정보를 질문으로 보완한다.

### Plan Generation
생성:
- 사업명
- 필요성
- 목적
- 추진계획
- 성과지표
- 기대효과
